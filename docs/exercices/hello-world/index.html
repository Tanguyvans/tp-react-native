Hello World

In this first exercise, you'll build an app that introduces core concepts through a simple counter app. We'll start by creating a structured layout with a header and main content area. Then, we'll add styled text components and implement interactive buttons with visual feedback. Throughout the exercise, we'll apply professional styling to create a polished user interface.

Here's what your final app will look like:

Step 1: run the project​

First, navigate to the exercise directory and install the dependencies:

cd exercises/01-hello-world
npm install

Then, run the project:

npx expo start

You'll see several options to run your app:

  • Press w to run in web browser
  • Press a to run on Android emulator
  • Press i to run on iOS simulator
  • Scan QR code with Expo Go app on your phone

If you have connection issues, try running with tunnel:

npx expo start --tunnel

Step 2: Create Hello-World page​

Your project is running, but it is empty. Let's add a simple page with a text component. Go to the app/index.tsx file and add the following code:

import { StyleSheet, Text, View } from "react-native";

export default function Page() {
  return (
    <View>
      <Text>Hello World</Text>
    </View>
  );
}

let's add some styles to the page.

const styles = StyleSheet.create({
  container: {
    flex: 1,
    alignItems: "center",
    justifyContent: "center",
  },
});

How can we link the styles to the page?

<View style={styles.container}>
  <Text>Hello World</Text>
</View>

Step 3: Add a counter​

Let's add interactivity to our app with a counter. We'll use React's useState hook to manage the counter state. The useState hook lets us track values that can change over time.

First, import the useState hook and add it to your component:

import { StyleSheet, Text, View, Pressable, SafeAreaView } from "react-native";
import { useState } from "react";

Wrap your main content with SafeAreaView:

export default function Page() {
  const [count, setCount] = useState(0);

  return (
    <SafeAreaView style={styles.container}>
      {/* Your content here */}
    </SafeAreaView>
  );
}

📱 Why SafeAreaView?​

SafeAreaView automatically adds padding to prevent content from being hidden by:

  • iPhone notches
  • Rounded corners

Now that we have a count variable, we can use it in the Text component.

<Text>{count}</Text>

Next, let's create buttons to control the counter. We'll use Pressable instead of the basic Button component because it offers better styling options and press feedback:

<View>
  <Pressable onPress={() => setCount(count + 1)}>
    <Text>Increment</Text>
  </Pressable>
  <Pressable onPress={() => setCount(count - 1)}>
    <Text>Decrement</Text>
  </Pressable>
</View>
<View>
  <Button title="Increment" onPress={() => setCount(count + 1)} />
  <Button title="Increment" onPress={() => setCount(count - 1)} />
</View>

The pressed state in our style allows us to show visual feedback when the user touches the button. We also prevent the counter from going below zero using Math.max(0, prev - 1).

Step 4: Style the App​

Let's style our app to make it look professional. We'll break down the styling into sections:

Header Styling​

header: {
  backgroundColor: '#007AFF',
  padding: 20,
  alignItems: 'center',
  shadowColor: "#000",
  shadowOffset: {
    width: 0,
    height: 2,
  },
  shadowOpacity: 0.1,
  shadowRadius: 4,
  elevation: 3,
},
headerText: {
  color: 'white',
  fontSize: 20,
  fontWeight: '600',
},

Welcome Section​

welcomeSection: {
  alignItems: 'center',
  marginVertical: 30,
},
title: {
  fontSize: 40,
  fontWeight: "bold",
  color: '#1a1a1a',
  marginBottom: 8,
},
subtitle: {
  fontSize: 18,
  color: '#666',
},

Card and Counter​

card: {
  backgroundColor: 'white',
  borderRadius: 16,
  padding: 24,
  width: '100%',
  maxWidth: 350,
  alignItems: 'center',
  shadowColor: "#000",
  shadowOffset: {
    width: 0,
    height: 2,
  },
  shadowOpacity: 0.1,
  shadowRadius: 4,
  elevation: 3,
},
counterSection: {
  alignItems: 'center',
  marginBottom: 24,
},
counterNumber: {
  fontSize: 48,
  fontWeight: 'bold',
  color: '#007AFF',
},

Button Styles​

buttonGroup: {
  flexDirection: 'row',
  gap: 16,
  marginBottom: 16,
},
button: {
  width: 60,
  height: 60,
  borderRadius: 30,
  justifyContent: 'center',
  alignItems: 'center',
  shadowColor: "#000",
  shadowOffset: {
    width: 0,
    height: 2,
  },
  shadowOpacity: 0.1,
  shadowRadius: 4,
  elevation: 3,
},
buttonPressed: {
  opacity: 0.8,
  transform: [{scale: 0.95}],
},

💅 Key Styling Concepts​

  1. Layout

    • Use flex: 1 for full height
    • alignItems and justifyContent for centering
    • flexDirection: 'row' for horizontal layouts

  2. Shadows

    • iOS: use shadowColor, shadowOffset, shadowOpacity, shadowRadius
    • Android: use elevation

  3. Spacing

    • Use consistent margins and padding
    • Group related elements with marginVertical or marginHorizontal

  4. Animations

    • Use transform and opacity for button feedback
    • Scale animations create a "press" effect

Try changing these values to create your own theme:

  • Change the primary color (#007AFF)
  • Adjust the shadow values
  • Modify the button sizes
  • Create new animations
